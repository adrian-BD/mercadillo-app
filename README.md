# Mercadillo de Muchavista - PWA

Aplicación web progresiva para el Mercadillo de Muchavista en Sant Joan d'Alacant.

## Características

- Hero dinámico con indicador de estado en tiempo real
- Mapa interactivo de Google Maps
- Directorio de puestos con filtros por categoría
- Ofertas relámpago en tiempo real
- Panel de administración para vendedores
- Navegación inferior mobile-first
- Diseño responsive con paleta mediterránea

## Tecnologías

- React + TypeScript
- Tailwind CSS
- Supabase (Base de datos)
- Vite
- Lucide React (Iconos)

## Desarrollo

```bash
npm install
npm run dev
```
